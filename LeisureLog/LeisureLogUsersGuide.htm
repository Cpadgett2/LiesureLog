1         Introduction

Leisure Log is the first application of its kind. It is designed to be a digital replacement for the standard logbook used by the Marine Corps for tracking where Marines go when they leave the barracks after working hours. The intent of this document is to provide the user(s) of Leisure Log a guide on the various functions within the application, instructions on how to use each function, as well as some information on the various error messages users may encounter.

2         How to use this Guide

This guide is designed with both step-by-step guidance and quick reference application in mind. New users will be able to navigate the program without any prior training; while users already familiar with the system can use it to quickly reference a particular function. As such, all job aids are organized in a logical order based on how the application is intended to be used. To enhance readability job aids are setup inside of tables. Each table has two columns, the left is the step number and the right are the instructions and pictures associated with that step. To facilitate the dual nature of this guide, we have placed hyperlinks in the table of contents as well as in the body. Throughout this guide, wherever a specific function is named in a body of text, the term will be hyperlinked to the exact location where that specific function can be found. If a certain function depends on the prerequisite completion of another function, the prerequisite function will also be hyperlinked in that section. Continue to Initial Configuration job aid.

3.2 Liberty Management

3.2.1 Liberty Check-Out

Step

Instructions

1

         On the Main page, Enter Marines DODID and click lookup

         Verify the Marines information matches their ID and Tier Card.

         If the Marine is not found, and you are presented with this error message:

         follow the Add Marine job aid

         If the Marines information has changed, such has their rank or tier card level, follow the Update Marine job aid before checking them out

2

         Click Add to Group, to put them in the liberty group:

         Repeat steps 1 and 2 for each Marine in the group.

3

         Enter destination and a contact number for the group. These fields are required.

4

         Click Check Out.

         All Marines in the group should populate in the log with a group ID

         The system date and time will auto-populate in the check-out column

         Check Out Successful will be displayed in the green box

5

         If Destination or Contact Number is not entered, you will be presented with one of the following dialogue boxes:

         Along with a Check Out Failure message in a red box

6

         Enter the required information and continue with the check out process for each group

7

         Some Marines are authorized to check out without a liberty buddy. If you try to check out a Marine without a liberty buddy and he or she is not the appropriate rank and tier level, you will be presented with the following error message:

8

         If you need to remove a Marine from a group before check out, select the Marines name and click Remove

9

Job End.

3.3 Admin Function: Marine Management

3.3.1 Add Marine

Step

Instructions

1

         From the Main screen click Admin and select Marine Management

2

         The Marine Options screen will be displayed:

        

3

         Enter DODID, Rank, Last Name, First Name, Middle Initial directly from the Marines ID. All fields are required except middle initial:

         Enter Tier Card level shown on the Marines Tier Card

         If they do not have a tier card yet, they are Tier 3

         Room number should be verified with barracks manager

4

         After all information has been verified and entered, click Add Marine

5

         A confirmation info dialogue box will be displayed, click OK

6

         Repeat steps 1-5 until all Marines have been added

         When you are finished click the X in upper right-hand corner

7

         You will be asked to Save Changes to the Marine File?

         Click Yes

* Note: Clicking No will close the Marine Options box without updating the data file, clicking Cancel will return you to the Marine Options form*

8

         When the Save to File dialogue box appears, double click the LeisureLog folder

         Then select the marinedata.csv and click Save to File

        

9

Job End.
