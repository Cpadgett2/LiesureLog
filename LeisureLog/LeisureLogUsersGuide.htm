1         Introduction

Leisure Log is the first application of its kind. It is designed to be a digital replacement for the standard logbook used by the Marine Corps for tracking where Marines go when they leave the barracks after working hours. The intent of this document is to provide the user(s) of Leisure Log a guide on the various functions within the application, instructions on how to use each function, as well as some information on the various error messages users may encounter.

2         How to use this Guide

This guide is designed with both step-by-step guidance and quick reference application in mind. New users will be able to navigate the program without any prior training; while users already familiar with the system can use it to quickly reference a particular function. As such, all job aids are organized in a logical order based on how the application is intended to be used. To enhance readability job aids are setup inside of tables. Each table has two columns, the left is the step number and the right are the instructions and pictures associated with that step. To facilitate the dual nature of this guide, we have placed hyperlinks in the table of contents as well as in the body. Throughout this guide, wherever a specific function is named in a body of text, the term will be hyperlinked to the exact location where that specific function can be found. If a certain function depends on the prerequisite completion of another function, the prerequisite function will also be hyperlinked in that section. Continue to Start Program / Login job aid.

3.1.1 Start Program / Login

Step

Instructions

1

Navigate to the application and double click on the Leisure Log icon. You should see a dialog box like the one depicted below which says Reading Master File

 

*Note: If you receive a message saying No Master File Found or Master File Corrupted, please notify the application administrator to obtain a new master file then Update Master File *

2

Once the application loads, you will be prompted to enter your DODID number and press enter. If you have logged onto the system previously your information will be displayed for your verification. Once you have verified your information is correct, click login.

 

*Note: If you have not previously logged in to Leisure Log, you will be prompted to enter your information before you can login*

3

Job End.

Continue to Liberty Checkout

3.1.2 Liberty Check-Out

Step

Instructions

1

Enter the Marines DODID into the text box, then verify the information with the Marines ID and Tier card.

*Note 1: If a Marine is not in the database, a dialogue box will ask if you would like to add the Marine. Click yes to Add Marine*

*Note 2: If a Marines information has changed, follow the Update Marine job aid before allowing the Marine to check-out*

2

Click add Marine, to put them in the liberty group. Repeat steps 1 and 2 for each Marine in the group.

3

Enter destination of the group. This field is required.

4

Click checkout. All Marines in the group should populate in the log below. Verify that the system date and time populates in the Date/Time Out field.

5

Job End.

Continue to Liberty Check in

3.1.3 Liberty Check In

Step

Instructions

1

When a Marine(s) return from liberty, locate the record by the names on their ID cards.

2

Click the check box next to each Marine.

3

Click the check-in button in the form located above the log

4

Verify that the system date/time populates in the Date/Time In field.

5

Job End.

3.2 Marine Management Functions

3.2.1 Add Marine

Step

Instructions

1

From the main page click on GUI component for Marine Management

2

Select Add Marine Tab

3

Ask Marine to present Identification and Tier Card

4

Enter DODID, Rank, Last Name, First Name, Middle Initial directly from the Marines ID. All fields are required except middle initial.

5

Enter Tier Card level shown on the Marines Tier Card

6

Use Marines name to verify then enter the room # from the room roster (provided by barracks manager)

7

Click Save.

8

Job End.

Marine can now be added to a Liberty Check-Out Group

3.2.2 Update Marine

Step

Instructions

1

From the main page click on GUI component for Marine Management

2

Ask Marine to present Identification and Tier Card

3

Enter DODID number from Identification Card and click find

4

Update information that has changed.

5

Click Save

6

Job End.

3.3 Administrator Functions

3.3.1 Update Master File

Step

Instructions

1

After obtaining the master file and starting the application described in Start Program / Login, click on the Administration pull down menu and click Import File to bring up the file browse dialogue box.

2

Click browse and navigate directory containing the master file. Click on the master file then click import.

3

When the Would you like to overwrite the file dialogue box appears, click yes. When confirmation message is displayed. Click okay.

4

Close and restart the application.

5

Job End.

Follow Start Program / Login procedure as before, then proceed to Liberty Check-out
